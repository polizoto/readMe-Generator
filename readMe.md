# readMe-Generator

This node app prompts the user to answer questions about project / app and will produce a professional ReadMe file.

## Features

The ReadMe file has the following structure:

- Title
- Badges
- screenshot (optional)
- Description 
- Table of Contents
- Installation
- Features
- Usage
- License (including a LICENSE file)
- Contribute
- Tests
- Contact
